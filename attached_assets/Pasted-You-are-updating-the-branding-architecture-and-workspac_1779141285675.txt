You are updating the branding architecture and workspace identity system for FilmCraft, a cinematic production operating system built by Studio di Gratia.

CORE CHANGE

“FilmCraft” is the platform/product name.

“Studio di Gratia” is NOT the platform name anymore.
It is now the personal studio/workspace identity of the current user.

Every creator, production company, filmmaker, collective, researcher, or studio using FilmCraft should be able to customize and rename their own workspace/studio environment.

The system architecture should now work like this:

FilmCraft → Workspace/Studio → Project

Examples:
• FilmCraft → Studio di Gratia → Ghost Theory
• FilmCraft → Neon Vale Pictures → The Last Signal
• FilmCraft → AK/OS → Homelines

BRANDING GOAL

The platform should feel like:
• a professional cinematic operating system
• infrastructure for filmmakers and creative studios
• modular and scalable
• personal and author-driven
• less like a generic productivity app
• more like a real production environment

Do NOT over-brand “FilmCraft” across every interface panel.
FilmCraft should function as the infrastructure layer, while the user’s studio/workspace identity becomes the emotional and visual center of the experience.

UI/UX CHANGES

Update the top navigation hierarchy:

Current:
Studio di Gratia → StarLife

New:
FilmCraft → Studio di Gratia → StarLife

Where:
• FilmCraft = platform
• Studio di Gratia = current workspace/studio
• StarLife = project

Add support for customizable workspace/studio names.

Each user should be able to:
• create a studio/workspace name during onboarding
• edit it later in settings
• optionally upload logo/avatar/banner
• define studio type:
  • Production Company
  • Independent Filmmaker
  • Creative Collective
  • Research Lab
  • Interactive Media Studio
  • Freelancer
  • Hybrid

IMPORTANT POSITIONING

The emotional identity belongs to the creator’s studio/workspace.
The technical identity belongs to FilmCraft.

FilmCraft should feel like the engine.
The workspace should feel like the filmmaker’s world.

Do not rewrite the entire visual system.
Refine the hierarchy and branding architecture while preserving the existing cinematic aesthetic and production OS direction already implemented.
